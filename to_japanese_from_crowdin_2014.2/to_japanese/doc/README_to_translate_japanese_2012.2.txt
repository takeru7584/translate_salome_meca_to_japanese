1.概要
このスクリプトは、Salome-Meca 2012.2に
日本語メニュー環境を追加するための変換実行環境です。
このスクリプトを用いて、日本語環境を追加することができます。

また、用意した日本語テーブルファイルを編集することで、
メニューのキーワードとなる日本語を編集し、
それを反映することもできます。

2.必要な実行環境
	2.1 必要条件
		2.1.1 Ubuntu Linux (Salome-Meca動作可能のもの)
		2.1.2 Salome-Meca 2012.2 (他のバージョンは未対応)
		2.1.3 qt4-dev-tools 
		    (スクリプトにてインストール可能（要インターネット）
		2.1.4 ruby1.9.1
		    (スクリプトにてインストール可能（要インターネット）
		2.1.5 インターネット接続環境
		    (2.3、2.4の導入に必要)
	
	2.2 推奨条件
		日本語テーブルファイルの編集に使用

		2.2.1 LibreOffice、またはOpenOffice
		2.2.2 テキストエディタ (geditで可)
		2.2.3 Linuxの知識 (コマンドラインでの作業)

3.インストールと実行準備
		3.1 translate_salome_meca_2012_2_to_japanese_a011.tgzをSalome-Mecaの
				インストール先にコピー

$ sudo cp translate_salome_meca_2012_2_to_japanese_a011.tgz /opt/SALOME-MECA-2011.2-LGPL
		
		3.2 translate_salome_meca_2012_2_to_japanese_a011.tgzを解凍

$ cd /opt/SALOME-MECA-2012.2-LGPL
$ sudo tar -xvzf translate_salome_meca_2012_2_to_japanese_a011.tgz

		3.3 （必要パッケージ未インストール時のみ）
		    スクリプト実行環境をインターネットよりインストール
				Ubuntu 11.04用にスクリプトを用意しております。
				（要インターネット接続）
			  以下のコマンドを実行してください。
				（Ubuntu 11.04 or DEXCS Salome 2011 A3のみ）

$ ./preinstall_exec_to_translate_japanese.sh



4.日本語化の実行
		4.1 日本語化スクリプトの実行

$ ./exec_translate_to_japanese.sh


5.コマンドラインオプション
Usage:./exec_translate_to_japanese.sh [option]

	-?, --help	
							コマンドラインオプションの説明を表示します。

	-d, --debug	
							デバッグモードで実行します。
							デバッグモードではログレベルは10になり、
							ログファイル出力モードでも標準エラー出力します。

	-f, --dic-file <dic_file_path(String)>
							翻訳辞書ファイルを指定します。

	-l, --log-level	<n(0<=n<=10>
							ログレベルを設定します。0は表示なし、10が全表示です。
							デフォルトは1です。

	-o, --log-file <log_file_path(String)>
							ログファイルモードで実行します。
							ログファイルを指定した場合、ログを標準エラー出力せず、
							ファイルに出力します。
							ただし、デバッグモードでは、ログファイルと
							標準エラー出力の両方に出力します。

	-q, --quiet
							実行時、ログを出力しません。（ログレベル0と同等）

	-u, --udic-file <user_dic_file_path(String)>
							翻訳ユーザ辞書ファイルを指定します。
							ユーザ辞書はユーザが翻訳辞書キーワードを追加できます。

	-v, --version
							日本語化開発スクリプトのバージョンを表示します。


6.日本語環境への設定変更方法
		5.1 File >> Preference を選択
		5.2 Language を jaに変更
		5.3 Salome-Mecaを再起動


7.日本語テーブルファイルの編集方法
		6.1 日本語テーブルファイル格納場所

$ ./to_japanese/dic/org/summary_translate_word.dic

		6.2 テーブルのフォーマット
			タブ区切りで以下の通りとなる。

対象qmファイル名	キーワード	フランス語	英語	日本語

		6.3 編集方法
			6.3.1 viエディタでの編集
						ターミナル上で編集が可能です。

			6.3.2 geditでの編集
						gui環境上でのみの動作ですが、
						操作方法はメモ帳と同じです。

			6.3.3 LibreOffice、OpenOfficeでの編集
						gedit等で全選択し、LibreOffice等に貼り付けると、
						各項目がセル毎に格納されるため、
						見た目が見やすく、容易に編集が可能。
						編集後、全選択し、geditでsummary_translate_word.dicに
						再貼り付けすることで編集作業完了。

			6.3.4 編集後の摘要作業
						日本語化スクリプトを実行

$ ./exec_translate_to_japanese.sh

8.メニューキーワードの確認方法
		7.1 File >> Preference を選択
		7.2 Language を raw keywordに変更
		7.3 Salome-Mecaを再起動
		7.4 再起動後に実行して表示されるメニューは
				キーワード表示となる
